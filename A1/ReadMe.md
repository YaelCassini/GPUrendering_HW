### 操作说明：

1. 按键1为ivory shader，按键2为gooch shader。
2. **wasdzc**按键可以控制相机上下左右移动的移动（表现为模型上下左右移动）。
3. **ijklm,**按键可以控制灯光上下左右前后的移动。
4. 空格可以控制模型的旋转。



### 实验环境：

1. Visual Studio 2019
2. OpenGL 4.5
3. freeglut 3.0.0-1.mp for MSVC
4. glew 2.1.0
